# Changelog

All notable changes to the "dot-art-in-code" extension will be documented in this file.

## [0.0.1] - 2026-03-18

### Added

- Initial release of the extension.
- Added Status Bar button to open the QuickPick menu.
- Added feature to insert built-in `.txt` dot art files.
- Added feature to insert a random dot art.
- Added smart language detection to automatically apply correct comment syntax (supports Python, Rust, C++, JS, TS, HTML, etc.).
- Added right-click context menu feature: "Save Selection as New Art".
- Added user setting `dotArtInCode.customFolder` to support loading and saving custom art files.
